# Claude Guidelines

## Commit Style

Follow the [gitmoji](https://gitmoji.dev) convention for all commits.

### Format

```
<emoji> <short description>

[optional body]
```

### Common emojis

| Emoji | Code                          | Use when                             |
| ----- | ----------------------------- | ------------------------------------ |
| ✨    | `:sparkles:`                  | New feature                          |
| 🐛    | `:bug:`                       | Bug fix                              |
| 🎨    | `:art:`                       | Improving structure / format of code |
| ♻️    | `:recycle:`                   | Refactoring code                     |
| 🔥    | `:fire:`                      | Removing code or files               |
| 📦    | `:package:`                   | Adding or updating packages          |
| 🚀    | `:rocket:`                    | Deploying stuff                      |
| 💄    | `:lipstick:`                  | UI / styling changes                 |
| 🔧    | `:wrench:`                    | Configuration files                  |
| 📝    | `:memo:`                      | Documentation                        |
| ✅    | `:white_check_mark:`          | Adding or updating tests             |
| 🔒    | `:lock:`                      | Security fixes                       |
| 🌐    | `:globe_with_meridians:`      | Internationalization                 |
| ⚡    | `:zap:`                       | Performance improvements             |
| 🎉    | `:tada:`                      | Initial commit                       |
| 🔀    | `:twisted_rightwards_arrows:` | Merging branches                     |
| ⬆️    | `:arrow_up:`                  | Upgrading dependencies               |
| ⬇️    | `:arrow_down:`                | Downgrading dependencies             |
| 🚑    | `:ambulance:`                 | Critical hotfix                      |
| 💡    | `:bulb:`                      | Adding comments to source code       |
| 🧪    | `:test_tube:`                 | Adding a failing test                |
| 🗃️    | `:card_file_box:`             | Database changes                     |
| 🚧    | `:construction:`              | Work in progress                     |

### Examples

```
✨ Add hero section with animated background
🐛 Fix mobile nav not closing on route change
💄 Update color palette to match brand guidelines
📦 Add framer-motion for animations
🔧 Configure Tailwind dark mode
```
